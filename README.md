# 🤖 Robotic Arm Design in SolidWorks

## About This Project

This repository contains my progress while designing a robotic arm in SolidWorks.

I started this project to improve my CAD design skills and gain a better understanding of how robotic arms are built and assembled. Instead of creating everything at once, I am designing each component step by step and documenting the entire process here.

The project includes individual part models, assembly files, and screenshots showing the development of the robotic arm from the first component to the final assembly.

---

## Why I Chose This Project

As a Mechatronics student, I wanted a project that would help me improve my SolidWorks skills while also introducing me to the mechanical design aspects of robotics.

This project gives me practical experience with:

- Part modelling
- Assembly creation
- Mates and constraints
- Mechanical design concepts
- CAD project organization
